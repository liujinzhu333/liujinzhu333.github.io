什么是 HTML？

HTML 是用来描述网页的一种语言。
  • HTML（HyperText Markup Language）是一种用于创建网页的标记语言。
  • HTML 不是一种编程语言，而是一种标记语言
  • 标记语言是一套标记标签 (markup tag)
  • HTML 使用标记标签来描述网页
  • HTML 文档包含了HTML 标签及文本内容
  • HTML文档也叫做 web 页面
